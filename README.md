# m-dex

Crystal API wrapper for [MangaDex](https://mangadex.org).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     m-dex:
       github: nedpals/m-dex
   ```

2. Run `shards install`

## Usage

